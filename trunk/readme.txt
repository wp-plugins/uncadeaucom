== Description ==

Ce plugin permet d'afficher 3 produits issus de la base UnCadeau.com en dessous de vos articles. Il est possible de monétiser ces liens en devenant partenaire du site UnCadeau.com


== Installation ==


1. Uploader le dossier 'uncadeau' vers votre dossier `/wp-content/plugins/`
2. Activer le plugin depuis le menu extension de Wordpress
3. Editer les préférence du plugin depuis le panneau 'réglage' de Wordpress
4. Lorsque vous éditer un article entrez les mots clef permettant de sélectionner les 3 produits à afficher
